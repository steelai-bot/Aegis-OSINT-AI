# Australian OSINT Data Sources

## Government Registers
- **ABR (Australian Business Register)**: https://abr.business.gov.au — ABN/ACN lookup
- **ASIC Connect**: https://connectonline.asic.gov.au — Company register
- **IP Australia**: https://search.ipaustralia.gov.au — Trademarks/patents
- **ACNC**: https://www.acnc.gov.au — Charity register
- **Government Directory**: https://www.directory.gov.au — Public servant directory

## Breach Databases (API)
- **Have I Been Pwned**: https://haveibeenpwned.com/API/v3 — Breach/paste search ($3.50/mo)
- **DeHashed**: https://dehashed.com — Leaked credential search (paid)
- **Intelligence X**: https://intelx.io — Archive search (free tier available)
- **LeakCheck**: https://leakcheck.io — Credential leak search (paid)
- **Snusbase**: https://snusbase.com — Database search (paid)
- **BreachDirectory**: https://breachdirectory.org — Free breach search

## Telegram Resources
- **TGStat**: https://tgstat.com — Channel analytics & search
- **Telemetr.io**: https://telemetr.io — Channel discovery
- **Combot**: https://combot.org — Group analytics

## Dark Web Search
- **Ahmia.fi**: https://ahmia.fi — .onion search engine (clearnet)
- **DarkSearch**: https://darksearch.io — Dark web search API
- **Torch**: https://torchsearch.xyz — Dark web search
- **Ransomware.live**: https://ransomware.live — Ransomware victim tracker

## Paste Sites
- **Pastebin**: https://pastebin.com — Paste search (pro API for scraping)
- **GitHub Gists**: https://gist.github.com — Code paste search
- **Rentry**: https://rentry.co
- **dpaste**: https://dpaste.org
- **JustPaste.it**: https://justpaste.it

## DNS & Domain Intelligence
- **Google DNS**: https://dns.google — DNS resolution API
- **crt.sh**: https://crt.sh — Certificate Transparency logs
- **SecurityTrails**: https://securitytrails.com — Domain/DNS history (API)
- **Shodan**: https://shodan.io — Device/service search
- **Censys**: https://censys.io — Internet-wide scanning

## Australian Phone Lookup
- **White Pages AU**: https://www.whitepages.com.au
- **Reverse Australia**: https://www.reverseaustralia.com
- **ACMA Number Register**: https://www.acma.gov.au

## Australian Email Domain Patterns
- `.com.au` — Commercial
- `.net.au` — Network
- `.org.au` — Non-profit
- `.gov.au` — Government
- `.edu.au` — Education
- `.id.au` — Individual
- `.asn.au` — Association
- `.csiro.au` — CSIRO
